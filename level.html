    Frequently Requested Answers

    In case of any confusion, here are some rules to make things clearer about how these algorithms have been implemented and how you should give your answer:

    1. Type your answer in the plain text box to both encrypt and check it. If it is correct, an appropriate message will display. If it is incorrect nothing will happen and the plain text will simply be encrypted
    2. You may find a result that is not the correct answer but encrypts to the same ciphertext as the challenge given, but just like in the real world this will not count
    3. You do not need any specific punctuation or capitalisation in your answer, it only compares raw letters
    4. The implented encryption algorithm does not care for uppercase or lowercase letters, it may preserve them or just convert the plain text to upper case before encrypting
    5. Most of the algorithms implemented here will only encrypt alphabet characters, but will state all the characters that they can encrypt
    6. If an algorithm requires a key, the key will be the same for all users, but resets every day
    7. Two alternative versions of the same cipher will not necessarily share the same keys
    8. The objective has been encrypted with the exact same algorithm as the type in box
